Donat un String, invirteix-lo.

**Input Format**

Una línia de text

**Constraints**

No hi ha.

**Output Format**

El text amb l'ordre dels caracters invertit.

**Sample Input 0**

    abcde

**Sample Output 0**

    edcba

**Sample Input 1**

    ab c d efgh

**Sample Output 1**

    hgfe d c ba

**Sample Input 2**

    cgctagcttagctaacg

**Sample Output 2**

    gcaatcgattcgatcgc
