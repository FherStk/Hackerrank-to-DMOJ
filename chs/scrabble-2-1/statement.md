Donada una paraula, obté la seva puntuació d'Scrabble.

    Letter                           Value
    A, E, I, O, U, L, N, R, S, T       1
    D, G                               2
    B, C, M, P                         3
    F, H, V, W, Y                      4
    K                                  5
    J, X                               8
    Q, Z                               10

**Input Format**

Una paraula

**Constraints**

\-

**Output Format**

La puntuació obtinguda

**Sample Input 0**

    hola

**Sample Output 0**

``` 
7
```

**Explanation 0**

`  h=4 o=1 l=1 a=1 `

**Sample Input 1**

    mundo

**Sample Output 1**

``` 
8
```

**Explanation 1**

`  m=3 u=1 n=1 d=2 o=1 `

**Sample Input 2**

    java

**Sample Output 2**

``` 
14
```

**Explanation 2**

`  j=8 a=1 v=4 a=1 `

**Sample Input 3**

    HACKERRANK
