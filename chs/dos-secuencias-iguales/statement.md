Donades dues seqüències de nombres, dir si són iguals.

**Input Format**

La entrada consta de dues seqüències.

Per a cada seqüència, el primer nombre  indica la quantitat de nombres
de dita seqüencia. A continuació ve la seqüència.

**Constraints**

\-

**Output Format**

si les seqüències són iguals

si les seqüències no són iguals

**Sample Input 0**

    3    100 200 300
    3    100 200 300

**Sample Output 0**

    true

**Sample Input 1**

    3    100 200 300
    4    100 200 300 400

**Sample Output 1**

    false

**Sample Input 2**

    5    1 2 3 4 5
    5    1 2 3 4 5

**Sample Output 2**

    true

**Sample Input 3**

    5    1 2 3 4 5
    5    1 2 3 4 4

**Sample Output 3**

    false

**Sample Input 4**

    3    23 34 45
    2    23 45

**Sample Output 4**

    false

**Sample Input 5**

    5    1 2 3 4 5
    1    5

**Sample Output 5**

    false
